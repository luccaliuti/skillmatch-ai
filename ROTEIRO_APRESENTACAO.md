# Roteiro de apresentação

## Abertura (30 segundos)
"Desenvolvi uma plataforma web de recrutamento com currículo cego, cujo objetivo é reduzir vieses iniciais no processo seletivo, ocultando dados pessoais e ranqueando candidatos por aderência às competências da vaga."

## O que demonstrar (1 a 2 minutos)
1. Mostrar o campo de descrição da vaga.
2. Mostrar o upload de currículos.
3. Clicar em **Analisar currículos**.
4. Mostrar o texto anonimizado.
5. Mostrar o ranking por **fit score**.

## Explicação técnica (30 segundos)
"O sistema foi desenvolvido em Python com Streamlit, funcionando como uma aplicação web. O algoritmo lê o currículo, remove dados sensíveis por regras simples e calcula a aderência da vaga com base em similaridade textual e palavras-chave."

## Fechamento (20 segundos)
"Como próximos passos, o projeto poderia evoluir com login, banco de dados, relatórios, integração com RH e inteligência artificial mais avançada para análise semântica."
